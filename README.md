## How to Provide Feedback

1. **Bug Report** — Something broken or not working as expected? [Report a bug](https://github.com/PruneLab/feedback/issues/new?template=bug_report.yml).

2. **Feature Request** — Have an idea to improve a product? [Suggest a feature](https://github.com/PruneLab/feedback/issues/new?template=feature_request.yml).

Please make sure to select the correct **Product** in the form so we can route your feedback quickly.

## Guidelines

- Search existing issues before creating a new one to avoid duplicates.
- One issue per topic — don't mix multiple bugs or features in one issue.
- Provide as much detail as possible (OS version, app version, steps to reproduce, screenshots).
- Be respectful and constructive.
